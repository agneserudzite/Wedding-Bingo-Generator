# Wedding Bingo Card Generator

A Python script to generate unique, print-ready Latvian Wedding Bingo cards using the ReportLab PDF library.

## Features
- Generates distinct cards with randomized prompts.
- Full support for Latvian special characters.
- Automated word-wrapping for grid text.
- Formatted for A4 page output.

## Prerequisites
- Python 3.x
- ReportLab library: `pip install reportlab`
- Font files: `HolidayFree.ttf`, `times.ttf`

## Installation
1. Clone the repository.
2. Place the required `.ttf` font files in the project root.
3. Install dependencies.

## Usage
Run the script via terminal:
```bash
python3 bingo_cards.py
